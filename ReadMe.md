## 下载 
### Now Playing
获取歌曲信息的软件，你需要下载 [Windows 本体](https://github.com/Widdit/now-playing-service/releases)，已知兼容2.0.0版本。
### Mira
简单来说，Mira 是个类似 Rainmeter、Wallpaper Engine和PPT 的东西。你需要[下载 Mira](https://www.miradesktop.com) 并使用下列链接导入本组件:

Vanilla 分支，Now Playing 标准组件，需要Mira0.4.5+。
```
https://github.com/Lumi083/now-playing-MiraUI/archive/refs/heads/vanilla.zip
```
main 分支，基于 Now Playing API 实现的组件。
```
https://github.com/Lumi083/now-playing-MiraUI/archive/refs/heads/main.zip
```
除了从链接导入，也可以下载后导入文件。
## 选择 Mira 的理由
#### 对于直播主
Mira 提供了直观的悬浮窗交互功能，图层、3d、投影等能力，并且可以直观地确认歌词组件的位置和状态。
#### 对于一般用户
Mira 可以兼容 Wallpaper Engine 等壁纸软件，尤其推荐和 Mira 的音频可视化系列组件搭配使用。
#### 修改组件
Mira 组件的开发非常简单，Now Playing Mira UI只用了几个小时就做好了。如果你有额外的需求，由于组件的源码可以完全独立地存在于一个html文件内，编辑和分发都更加简单，你甚至可以直接丢给GPT让他想办法。
